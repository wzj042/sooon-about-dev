素问显示文章标题

自动将浏览器标签页标题设置为当前参考源的标题

安装脚本

请先确保已安装 Tampermonkey 扩展，点击下方按钮即可一键安装。

🚀 一键安装

点击按钮后将跳转到 Tampermonkey 安装页面

功能说明

  • 自动检测页面中的参考源标题（格式：#标题#）
  • 将浏览器标签页标题更新为参考源标题
  • 支持动态内容加载，自动监听 DOM 变化
  • 支持 https://sooon.ai/** 所有页面

使用方法

  1. 安装脚本后，访问 Sooon.ai 上的任意页面
  2. 当页面包含参考源标题时，浏览器标签页标题会自动更新
  3. 无需任何操作，脚本会自动工作

技术细节

工作原理
  • 使用 MutationObserver 监听 DOM 变化
  • 查找包含 #标题# 格式的元素
  • 提取标题文本并更新 document.title
  • 包含轮询机制，确保动态加载的内容也能被检测到

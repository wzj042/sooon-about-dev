Sooon.ai 多功能链接复制器

复制链接时，可以选择复制 Markdown 格式或纯链接格式，以及来源和素问的链接

⚠️
注意：当前在非主页下获取的链接可能存在异常，如顺序阅读下打开的文章可能会意外的复制顺序阅读的当前页面链接，考虑修复中

安装脚本

请先确保已安装 Tampermonkey 扩展，点击下方按钮即可一键安装。

🚀 一键安装

点击按钮后将跳转到 Tampermonkey 安装页面

功能说明

  • 在文章弹窗的工具栏中添加 "素" 和 "源" 两个快捷复制按钮
  • "素" 按钮：复制素问链接（Markdown 格式或纯链接）
  • "源" 按钮：复制来源链接（Markdown 格式或纯链接）
  • 支持切换复制模式：
    • Markdown 模式：复制为 Markdown 格式链接，如 [《标题》](url "问题")
    • 纯链接模式：直接复制 URL
  • 通过 Tampermonkey 菜单命令切换复制模式
  • 支持 https://sooon.ai/** 所有页面

使用方法

  1. 打开 Sooon.ai 上的任意文章弹窗
  2. 在底部工具栏中可以看到新增的 "素" 和 "源" 按钮
  3. 点击 "素" 按钮复制素问链接
  4. 点击 "源" 按钮复制来源链接
  5. 点击 Tampermonkey 图标，选择菜单中的 "切换复制模式" 来切换 Markdown/纯链接格式

技术细节

复制格式说明
  • Markdown 模式：
    • 素问链接：[《标题》](素问URL "问题")
    • 来源链接：[标题](来源URL "问题")
  • 纯链接模式：直接复制 URL 地址

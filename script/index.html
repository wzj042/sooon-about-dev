油猴插件基础说明

1. 安装 Tampermonkey

在使用这些脚本之前，你需要先安装 Tampermonkey 浏览器扩展。

  • 【推荐】Chrome：Chrome 网上应用店
  • 【推荐】Edge：Microsoft Edge 加载项
  • 蓝奏云下载

1.1 如何固定 Tampermonkey 到工具栏

安装 Tampermonkey 后，建议点击拓展程序（拼图图标）将 Tampermonkey 固定到工具栏，方便后续使用：

  1. 点击浏览器工具栏中的 Tampermonkey 图标
  2. 选择 "固定到工具栏"

2. 如何添加脚本

安装 Tampermonkey 后，按照以下步骤添加脚本：

  1. 点击浏览器工具栏中的 Tampermonkey 图标
  2. 选择 "创建新脚本..." 或 "添加新脚本..."
  3. 在脚本编辑器中，删除默认内容
  4. 打开要安装的脚本文件（.user.js 文件），复制全部内容
  5. 将复制的内容粘贴到 Tampermonkey 脚本编辑器中
  6. 按 Ctrl+S（或 Cmd+S）保存脚本
  7. 脚本会自动启用，刷新目标网页即可生效
开启开发者模式（可选）

如果需要手动管理脚本文件，可以通过扩展管理页面开启浏览器的开发者模式：

  1. 访问扩展管理页面：
    • Chrome: chrome://extensions/
    • Edge: edge://extensions/
  2. 在页面右上角或左侧找到并开启 "Developer mode"（开发者模式） 开关

📸 素问打卡奖励截图工具

一键生成打卡奖励分享长图

🚀 一键安装

点击按钮即可自动安装到 Tampermonkey

🎯 使用流程

  1. 在素问完成每日打卡，获得打卡奖励
  2. 等待打卡奖励弹窗出现
  3. 点击右下角 "生成分享长图" 按钮
  4. 截图将自动下载为 PNG 图片

🎯 智能检测

自动检测打卡奖励弹窗，只在需要时显示截图按钮，不影响正常浏览。

📷 长图生成

使用 html2canvas 技术，完整截取奖励内容区域，生成高质量长图。

⚡ 一键操作

点击按钮即可生成并自动下载，无需手动截屏和编辑，方便分享。

🎨 高清输出

2倍分辨率输出，白色背景，确保截图清晰美观，适合社交媒体分享。

📖 安装使用说明

  1. 确保已安装 Tampermonkey 浏览器扩展
  2. 点击上方 一键安装 按钮
  3. 在弹出的 Tampermonkey 页面中点击 安装
  4. 访问 https://sooon.ai 完成每日打卡
  5. 打卡奖励弹窗出现后，点击右下角绿色按钮即可生成长图

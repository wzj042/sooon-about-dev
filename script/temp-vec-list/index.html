📜 素问聚焦顺序阅读

优化素问阅读体验的油猴脚本

🚀 一键安装

点击按钮即可自动安装到 Tampermonkey

⚠️ 列表闪烁问题

当前存在列表闪烁问题，滑动列表到一定深度时会出现列表无法下滑、闪烁的情况，建议只看前几天的内容。

📏 宽度自适应

将瀑布流布局转换为垂直列表，最大宽度 900px 居中显示，解决宽度不足问题。

⏰ 时间排序

按发布时间智能排序：今日 → 昨天 → 天前，让你不错过最新内容。

🎛️ 模式切换

支持通过油猴菜单在列表模式和瀑布流模式之间切换。

📖 使用说明

  1. 确保已安装 Tampermonkey 浏览器扩展
  2. 点击上方 一键安装 按钮
  3. 在弹出的Tampermonkey页面中点击 安装
  4. 访问 https://sooon.ai/home/read/feed 即可自动生效
  5. 如需切换模式，点击扩展图标 → 油猴菜单 → 选择切换选项

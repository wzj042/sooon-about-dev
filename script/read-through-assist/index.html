功能说明

1. 键盘导航

  • A / D：在文章列表中左右切换（上一篇文章/下一篇文章）
  • F：打开当前聚焦的文章
  • W / S：在文章弹窗中上下滚动内容
  • Q：关闭文章弹窗，或在列表页面时翻到上一页
  • E：翻到下一页

2. 文章过滤

  • 根据阅读量自动隐藏已读文章
  • 可设置过滤阈值（FC），隐藏阅读量大于等于该值的文章
  • 显示阅读进度百分比
  • 支持暂停过滤（设置 FC 为 0）

3. 自动翻页

  • 当列表为空时自动翻页
  • 可设置翻页方向（上一页/下一页）
  • 可设置检测间隔（1-60 秒）

4. 页面状态保存

  • 自动保存当前页面、排序方式等状态
  • 刷新后自动恢复到上次浏览的页面

使用方法

基础操作

  1. 访问 Sooon.ai 的已发布文章列表页面
  2. 使用键盘快捷键进行导航和操作
  3. 当前聚焦的文章会显示橙色高亮边框

配置选项

点击 Tampermonkey 图标，在菜单中可以配置：

  • 🔍 设置过滤阈值 (FC)：设置阅读量过滤阈值，0 表示暂停过滤
  • 🔄 设置自动翻页方向：选择自动翻页时是向前还是向后
  • ⏱️ 设置自动翻页检测间隔：设置列表为空后多久自动翻页（1-60 秒）

使用场景

  • 快速浏览：使用键盘快捷键快速浏览大量文章
  • 筛选未读：通过设置过滤阈值，只显示未读或低阅读量的文章
  • 批量整理：结合键盘导航和编辑功能，快速为文章添加标签
  • 自动处理：开启自动翻页，让脚本自动处理已读文章

注意事项

  • 键盘快捷键仅在非输入框区域生效
  • 过滤功能会隐藏符合条件的文章，但不会删除
  • 阅读进度统计基于总文章数和已过滤文章数
  • 页面状态保存在浏览器本地存储中

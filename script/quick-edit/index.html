素问快捷编辑

在 Sooon.ai 文章弹窗中注入快捷编辑按钮，恢复直接编辑和关闭的体验

安装脚本

点击安装脚本

点击后 Tampermonkey 会自动提示安装

本地开发说明

如果你想在本地测试脚本，需要通过 HTTP 服务器访问：

  1. 在项目根目录运行 HTTP 服务器： 
    # Python 3
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
  2. 在浏览器中访问：
    http://localhost:8000/script/quick-edit/index.html
  3. 点击"安装脚本"按钮即可触发 Tampermonkey 安装提示

注意：直接打开本地 HTML 文件（file:// 协议）时，需要在扩展管理中开启"允许访问文件网址"

前置要求

1. 安装 Tampermonkey

在使用这些脚本之前，你需要先安装 Tampermonkey 浏览器扩展。

  • Chrome：Chrome 网上应用店
  • Edge：Microsoft Edge 加载项

2. 开启开发者模式

通过扩展管理页面开启浏览器的开发者模式：

  1. 访问扩展管理页面：
    • Chrome: chrome://extensions/
    • Edge: edge://extensions/
  2. 在页面右上角或左侧找到并开启 "Developer mode"（开发者模式） 开关

功能说明

  • 在文章弹窗的工具栏中添加 ✏️ 快捷编辑按钮
  • 点击后自动打开"我的编辑"界面
  • 编辑完成后自动关闭所有相关弹窗
  • 支持 https://sooon.ai/* 所有页面
